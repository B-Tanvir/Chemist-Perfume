
# ChemistPerfume

A perfume warehouse management website


## Features

- Warehouse Management
- Athentication
- Responsive for mobile and tablet
- Simple design


## Tech Stack

**Client:** React, TailwindCSS, React router, React Firebase Hooks, Toastify

**Server:** NodeJS, Express, MongoDB, Firebase
